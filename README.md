# CBP 2023 Industry Analysis

## Overview
This project analyzes the 2023 County Business Patterns (CBP) U.S. data to understand the business landscape by industry across the United States. Key metrics include number of establishments, total employees, and payroll.

The goal is to identify which industries are the largest employers, which sectors have the most businesses, and where payroll is highest.

## Data Source
- County Business Patterns (CBP) U.S. File, 2023  
- Source: [U.S. Census Bureau](https://www.census.gov/programs-surveys/cbp.html)

## Tools Used
- Microsoft Excel for data cleaning, pivot tables, and analysis  
- PDF for summary report creation  

## Project Files
- `data/CBP_2023_US_File_Cleaned.xlsx` — Cleaned and filtered dataset ready for analysis  
- `report/CBP_2023_Summary_Report.pdf` — Summary report of findings and recommendations  

## Key Findings
- Healthcare and Social Assistance is the largest employer nationwide with over 44 million employees.  
- Retail Trade leads in the number of establishments with over 2 million.  
- Manufacturing and Finance have fewer establishments but higher payrolls, suggesting larger companies with higher wages.

## Recommendations
- Investors should target healthcare and finance for growth potential.  
- Policymakers should focus on retail for employment support.  
- Business strategists can explore industries with high payrolls but fewer establishments.

## How to Use
Open the Excel file to explore data with pivot tables and charts. Review the PDF report for detailed insights and business implications.
